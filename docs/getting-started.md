# Getting started

Questa pagina spiega come iniziare con il progetto.

## Requisiti

- Python 3.8+
- pip

## Installazione

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Prima build

```bash
mkdocs build
```

o per sviluppo:

```bash
mkdocs serve
```
